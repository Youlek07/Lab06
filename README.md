# Lab06 – Side Bar w Android Studio

## Opis aplikacji
Aplikacja została wykonana jako projekt na zaliczenie tematu Lab06.
Zawiera ona menu boczne, czyli inaczej Sidebara, przy użyciu komponentu Navigation Drawer. 
Umożliwia on przełączanie między różnymi aktywnościami, bądź w tym przypadku fragmentami.


## Aktywności/Fragmenty, funkcjonalności Sidebara w aplikacji

### 1. Home
Ekran główny z tytułem i krótkim opisem.

### 2. About
Statyczna strona z informacjami o mnie (autorze) - imię i nazwisko.

### 3. ZSEGW
Fragment zawierający komponent WebView, który otwiera stronę internetową szkoły.

### 4. Exit
Po prostu zamyka aplikację.
